# pkg (pacman wrapper)

I can never remember [pacman](https://wiki.archlinux.org/title/Pacman)'s flags, and I don't always want to install a proper pacman wrapper like [pamac-cli](https://github.com/manjaro/pamac-cli). So this is a basic shell script wrapper for the common use-cases. It's designed to feel familiar to users of Debian's [`apt`](https://en.wikipedia.org/wiki/APT_(software)) and FreeBSD's [`pkg`](https://wiki.freebsd.org/pkg).

See [the shell script](./pkg) for usage info.
